# Documentación – Card Profile

Esta carpeta contiene la documentación del proyecto para poder entender y mantener el desarrollo de la Card Profile.

## Contenido

| Documento | Descripción |
|-----------|-------------|
| [**card-profile-especificacion.md**](./card-profile-especificacion.md) | Especificación completa: requerimientos, colores, tipografía, espaciados, estructura de la card, props del componente y criterios de implementación. Es la referencia para desarrollar o replicar la card. |
| [**card-profile-implementacion.md**](./card-profile-implementacion.md) | Qué se implementó en código: dónde está cada archivo, props del componente, estructura del JSX, iconos, colores usados y próximos pasos sugeridos. |

## Orden recomendado de lectura

1. **card-profile-especificacion.md**: leer antes de tocar código; define qué se debe construir y con qué criterios visuales y técnicos.
2. **card-profile-implementacion.md**: leer después para entender qué se hizo (componente, App, datos de ejemplo) y cómo seguir desarrollando.
